Jagatbhav
Jagatbhav is a modern, AI-powered knowledge-sharing ecosystem designed for writers, researchers, educators, and readers. It provides a seamless, secure, and multilingual (English/Hindi) environment for digital publishing and content discovery.

🚀 Key Features
Multilingual Support: Seamlessly toggle between English and Hindi content.

AI-Powered Assistant: "Jaishri," our AI content discovery and writing assistant.

Writer & Guest Dashboards: Easy-to-use tools for publishing blogs, articles, and research.

E-Book Marketplace: A dedicated section for purchasing and previewing digital assets.

PWA Ready: Installable directly to your mobile device for offline-first access.

Premium Design: A minimalist, dark-themed interface built with a focus on reader experience.

Zero-Cost Infrastructure: Hosted via GitHub Pages, ensuring the platform remains free, secure, and accessible.

🛠 Tech Stack
Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+).

Architecture: Mobile-first, responsive, and PWA-compliant.

Deployment: GitHub Pages.
